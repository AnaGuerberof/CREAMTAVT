# CREAMTAVT
Data from the CREAMT project that explores the reception of subtitles in different translation modalities (human translation, human post-editing and machine translation).
This data is explained in the article “A Spanish version of EastEnders”: a reception study of a telenovela subtitled using MT written by
Ana Guerberof-Arenas, University of Groningen, Joss Moorkens, Dublin City University and David Orrego, Warwick University.
This will be published in Jostrans in January 2024.
